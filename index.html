Competitive Analysis

First, we want to research other websites with aggregator functions. I chose 3 websites with a favoriting function. I chose to compare Etsy, Sephora, and Amazon's favoriting functionality and to outline the differences in a comparison chart.

Sephora: https://www.sephora.com/

Summary

Etsy's Favorites List: Etsy’s favorite list is the most basic. You can only click into the items from the favorites list and there is no one-click button to add the item to the cart. This leads to less users actually buying the item from the favorites list, because there is no visual incentive. You can search through items, but, in comparison, an add to cart button would have been more helpful than the search.

Sephora Loves List: Sephora’s “loves” list is more catered for users. Unlike Etsy, they have an easy “Add to Basket” button. This makes it easy for users to buy the item from the liked list. Though the Sephora list doesn’t show the date added, you can filter by recently added. The filter is a trade-off from the search in Etsy and Amazon, where both could be used to find a specific item.

Amazon Shopping List: Amazon’s shopping lists have the most detailed functionality. There are multiple shopping lists you can add, and you can name the lists. This is a good idea, because it allows users to navigate to self-categorized liked products (e.i. makeup, fashion, home goods). Also, Amazon has an immediate undo option after you click remove, which is more convenient than a second confirmation click to remove like Etsy, which requires extra work.

Takeaways

For my aggregator website, I want to implement similar features from the websites researched:

  • Sort items by price (high to low, low to high) and by name (A to Z, Z to A) like Sephora
  • One-click remove like Amazon and Sephora which makes it convenient to delete an item
  • Option to remove items from the aggregated list all at once like Amazon

Final List of Features

  • Sorting Feature: can sort by category of item (nut butter, jam, topping) and by flavor (savory, sweet)
  • Filtering Feature: can filter by price (high to low, low to high) and name (A to Z, Z to A)
  • Stackable Sort and Filter: sort and filter can be layered, existing sort applied when filter changes and vice versa
  • Aggregator Section: cart with items displayed on right, items also added to sandwich visual
  • Updated Count of Items in Aggregator: total count of items in cart and total price displayed at bottom cart
  • Reset functionality: can reset sort and reset filter within dropdown menu
  • 16 Item Cards: item image, item name, item category, item flavor, item price all displayed
  • Button Adds and Removes from Aggregator: button to "Add" to cart from item card, "X" to remove from cart
  • Clear Cart: "Trash Can" button to remove all items from cart
  • Responsive: website responsive to desktop, tablet, and mobile phone devices
  • AI-Generated Images: all images are generated with Prodia AI and edited with Photoshop to match aesthetic
  • Animation: items "laid" down with animation onto sandwich visual

List of Future Features

One shortcoming of the app is that there is no way for users to mark when they're done building their sandwitch. So, potential features to add could be:
